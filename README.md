# Free DevOps Labs

A list of every free lab, sandbox, and practice environment I could find for learning DevOps and Cloud engineering. No credit card needed, no "free trial" that charges you after 7 days. Actually free.

I got tired of googling "free kubernetes lab" every time I wanted to practice something, so I put this together. If you know of one I missed, open a PR.

## Kubernetes

| Platform | What you practice | How long it's free |
|---|---|---|
| [KillerCoda](https://killercoda.com) | 200+ scenarios, great for CKA/CKAD prep | Unlimited |
| [Play with Kubernetes](https://labs.play-with-k8s.com) | Multi-node cluster right in your browser | 4 hours per session |
| [Killershell](https://killershell.io) | Exam simulator that feels like the real thing | 2 hours free |
| [k3s](https://k3s.io) | Full cluster on your laptop in 30 seconds | Forever |
| [minikube](https://minikube.sigs.k8s.io) | Single-node local cluster | Forever |
| [kind](https://kind.sigs.k8s.io) | K8s clusters running inside Docker containers | Forever |
| [Google K8s Engine](https://cloud.google.com/kubernetes-engine/pricing) | 1 free Autopilot cluster | Forever (you pay for nodes) |
