# Contributing to free-devops-labs

Thank you for helping make DevOps learning accessible to everyone!

## How to Contribute

1. Fork this repository
2. Add your resource to the correct section
3. Keep the table format consistent
4. Open a Pull Request

## What We Accept

- Free labs and sandboxes only
- Must be working as of the contribution date
- Must not require payment for basic access
- Must be relevant to DevOps/Cloud topics

## Table Format

```markdown
| [Platform Name](URL) | What You Practice | Free Limit |
```

## What We Do NOT Accept

- Paid courses or platforms without a meaningful free tier
- Referral/affiliate links
- Outdated or broken resources
- Content unrelated to DevOps/Cloud engineering
